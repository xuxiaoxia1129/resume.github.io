FlowMindLab / Obsidian 个人知识库 ​

FlowMindLab 是我的 Obsidian 个人知识库和长期学习系统，线上站点为 flowmindlab.cn。它以 Obsidian 仓库作为内容源，通过 Quartz 技术发布为公开知识站，目前主要整理 知识管理、AI、机器学习、Python、MySQL、紫微斗数 等内容，未来会继续把机器学习等应用场景写成自己的理解和实践心得。

Obsidian本地长期知识库、个人记录和主题笔记管理
5 类 MOCAI / 机器学习、MySQL、Python、知识管理、紫微斗数
22 个内容文件当前公开站内容层的页面和主题入口
flowmindlab.cn已上线的公开知识站入口

项目定位 ​

FlowMindLab 是一张持续生长的个人知识地图。它的核心首先是 Obsidian 仓库中的知识库内容，Quartz 负责把这套 Markdown 知识结构发布成可访问的网站。

博客更适合发布完成度较高的文章，但我的很多内容处在持续学习和持续迭代中：知识管理方法、个人记录、AI 工具使用、机器学习理解、Python 自动化、MySQL 工作语法、Obsidian 方法、紫微斗数系统化学习等。如果只用普通文章结构，很难表达主题之间的关联。

所以我选择用 Obsidian 管理本地知识仓库，用 Quartz 将适合公开的部分发布为网页，让个人学习、项目复盘、长期记录和作品集形成连接。

项目背景 ​

我长期需要处理几类信息：

  • 工作中沉淀的数据产品方法、SQL、BI、项目复盘。
  • AI 和编程学习过程中的工具、概念和实践。
  • 紫微斗数、传统文化和 AI 解读系统相关知识。
  • 个人项目、简历网站、证券考试工具等过程文档。
  • 个人记录、阶段复盘和可以公开展示的长期能力证据。

如果这些内容只留在本地，外部看不到；如果全部写成文章，又会丢失知识之间的连接。FlowMindLab 的目标，是在“个人笔记”和“公开作品”之间搭一座桥。

项目目标 ​

  1. 建立一个长期可维护的个人知识库。
  2. 用 MOC 和双链组织主题，减少笔记散落。
  3. 将部分学习内容和项目过程发布为公开网页。
  4. 与简历网站形成互补：简历网站展示项目结果，知识库展示长期学习、个人记录和思考过程。
  5. 让知识管理、AI、机器学习、数据产品、紫微斗数等长期方向有统一入口。
  6. 未来持续补充机器学习等应用场景的学习心得和实践理解。

MOC 与知识地图 ​

FlowMindLab 的重点在于通过 MOC 组织入口，让主题之间形成可导航关系。

MOC内容方向价值
AI / 机器学习 MOCAI 工具、模型理解、学习笔记、应用场景和实践记录支撑 AI 产品、自动化能力和后续机器学习心得输出
MySQL MOC工作中常用 SQL、查询场景和数据处理方法支撑数据产品和 BI 分析
Python MOC自动化脚本、数据处理、工具搭建支撑个人项目和效率工具
知识管理 MOCObsidian、Quartz、笔记生命周期、发布流程支撑长期知识系统
紫微斗数 MOC紫微知识、规则、AI 解读系统相关内容支撑紫微 AI 项目

MOC 的作用，是让访问者从主题入口进入知识，避免陷入零散笔记。

工作流设计 ​

知识库里还沉淀了几个工作流页面：

工作流作用
note-lifecycle管理一条笔记从捕捉、整理、链接到发布的过程
obsidian-capture-workflow管理日常输入、资料收集和初步整理
quartz-publishing-checklist管理公开发布前的检查流程

这些工作流让知识库同时呈现结果和维护过程。

数据如何驱动 ​

FlowMindLab 里的“数据”指知识节点和链接关系。

text
学习资料 / 项目过程
  -> Markdown 笔记
  -> 标签 / 双链 / MOC
  -> 公开知识站
  -> 长期检索与复用

数据驱动体现在：

  • 用目录和 MOC 控制知识入口。
  • 用双链表达概念之间的关系。
  • 用资源库区分资料和个人理解。
  • 用工作流保证内容从收集到发布有稳定过程。
  • 用公开站点让学习资产变成可被访问的能力证明。

成果与价值 ​

  • 建立了基于 Obsidian 仓库的个人知识库，并通过 Quartz 技术发布为公开知识站：www.flowmindlab.cn。
  • 搭建首页、MOC、主题笔记、资源库和工作流等公开内容结构。
  • 将知识管理、AI、Python、MySQL、机器学习和紫微斗数放入统一知识地图。
  • 让个人学习过程、项目沉淀和简历作品集之间形成长期连接。
  • 展示了我在 Markdown 内容管理、知识架构和个人系统搭建上的能力。

个人能力沉淀 ​

FlowMindLab 对我来说是长期学习和个人记录的基础设施。它让我把“学过什么、做过什么、怎么思考、如何复用”沉淀下来，也让我能把个人能力从一次性项目扩展为可持续生长的知识系统。
