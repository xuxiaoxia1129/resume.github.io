## 项目拆解

| 项目 | 类型 | 数据价值 |
|---|---|---|
| [一房一万新房小程序](/projects/lansi/yifangyiwan-new-home) | To C 小程序 / 官网 | 用楼盘、价格、认购、摇号、地图、内容和行为数据支撑购房决策与私域转化；公开入口：[一房一万官网](https://www.realtynavi.com/) |
| [小房书](/projects/lansi/xiaofangshu-res) | To B 二手房数据产品 | 用成交、市占率、楼盘、门店和权限数据帮助中小中介经营 |
| [城市地图 / CityMap](/projects/lansi/citymap) | To B 决策系统 | 用 GIS、多源房地产数据和城市指标支撑房企拿地与市场研判；公开产品页：[城图介绍](http://www.citywavedata.com/productIntroChengTu) |
| [数据中心 / RAS / 基价模型](/projects/lansi/data-center-ras) | 数据平台 | 用价格模型、指数体系和底层数据支撑多条产品线 |
| [红盘系统](/projects/lansi/red-house-system) | 房企营销分析 | 用认购、定价、户配、销售和舆情数据支持新盘操盘 |
| [挂牌查价](/projects/lansi/listing-price-check) | To C 查询工具 | 用二手房核验价、挂牌价和订阅数据形成高频查询工具 |

## 代表成果

- 一房一万新房小程序累计用户 80 万+。
- 一房一万挂牌查价在 2021 年 7-8 月快速获得 2 万+用户。
- 小房书服务上百家付费中小中介门店。
- 澜思 / 一房一万公开合作伙伴覆盖华润置地、龙湖、金地集团、大华地产、万科、华发股份上海公司、绿地控股、保利地产、中海地产、上海地产集团、招商蛇口等地产行业客户。
- 城市地图数据决策分析系统获得计算机软件著作权。

## 能力沉淀

- 长期产品迭代：从 0 到 1 到多版本优化。
- 行业数据理解：楼盘、价格、成交、认购、板块、土地、客户、经纪公司业务。
- 数据产品表达：把复杂数据转成地图、图表、看板、搜索、榜单和决策工具。
- 指标与埋点：用访问、留存、转化、认购、订阅、进群等数据支持产品复盘。
- To C / To B 场景切换：既做用户决策工具，也做经纪公司和房企的业务系统。
